# Typescript React Jest Quick Start
## 클론
```
git clone https://github.com/QMCHE/typescript-react-jest-quick-start.git
```
## 사용법
```
// 모듈 설치
npm i

// 개발 서버 시작
npm start

// 빌드
npm build

// 테스트
npm test
```
## Dependencies
+ react
+ react-dom
+ react-router-dom
+ react-redux
+ redux
+ styled-components
